# Linux.do LDC 打赏助手

一个为 Linux.do 社区添加 LDC 积分打赏功能的油猴脚本。

## 安装

[点击安装](https://greasyfork.org/zh-CN/scripts/562047-linuxdo-%E6%89%93%E8%B5%8F%E5%8A%A9%E6%89%8B)

## 脚本用处

能够向任意楼层用户打赏（除了自己），而不需要对方配置任何信息（如果对方没有激活LDC除外）。

## 使用方法

- 首先去LDC站点，创建应用，获取`Client ID`和`Client Secret`。详情可以参考[官方文档](https://credit.linux.do/docs/how-to-use) 3.2章节。
- 按照油猴脚本并开启执行脚本权限（这一步就不教学了，自行搜索）。
- 点击上方连接安装打赏脚本，并开启脚本运行。
- 刷新L站页面，首次访问会提示输入`Client ID`和`Client Secret`，保存即可。
- 配置完毕，现在每个楼层原来的点赞按钮旁边都会多出一个打赏按钮了，快来试试 `v我50` 吧。

## 单号规则

打赏时会按照一定的规则生成业务单号。你可以在LDC的后台记录查询到对应打赏订单的业务单号。  
具体规则如下：  
LDR_T{topicId}_P{postId}_{timestamp}_{random4digits}  
这样通过单号能定位到打赏的帖子以及楼层，还有时间。  
比如 LDR_T789409_P1_1767983344195_8511 这个单号，解析出来：  
帖子ID：789409  
楼层ID：1  
那么拼接L站帖子链接就是：https://linux.do/t/topic/789409/1
